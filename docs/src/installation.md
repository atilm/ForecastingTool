# Installation

## Build from source

Prerequisites:

- Rust toolchain (stable)
- On Linux, system font libraries for plotting (fontconfig)

Build:

```bash
cargo build --release
```

The binary will be available at:

- Linux/macOS: `target/release/forecasts`
- Windows: `target/release/forecasts.exe`

## Using release binaries

Tagged releases publish Linux and Windows binaries in GitHub Releases.

Open the repository Releases page and download the asset for your platform.
